© 2026 Document Translator

  1. Upload your .pptx or .pdf document using the drag-and-drop area or the file browser.
  2. Select the source language of your document and the target language you want to translate into.
  3. Click the Translate button to start the AI-powered translation process.
  4. The system extracts text from each slide while mapping its position and formatting.
  5. Google Cloud Translation API processes the text, preserving context and grammar.
  6. Download your fully translated document with the original slide layout intact.

About

Document Translator is a web application that translates PowerPoint presentations into other languages using AI-powered translation. It preserves the original slide layout, fonts, colors, and formatting — only replacing the text content.

Built with a FastAPI backend and Google Cloud Translation API, it supports a wide range of languages and handles structured slide content with precision, making it ideal for students, educators, and professionals who need multilingual presentations.

This project was created as part of the CSC 4351 Capstone course by Adwaita Boralkar, Jaelany Glasgow, Jay Patel, and Mohamed Sharif.
